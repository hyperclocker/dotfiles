The **Terminal User Interface**  

# Software
- Text editor: vim  
- Music player: cmus  
- File manager: vifm && ranger  
- Web browser: w3m
- Mail client: mutt
- To-Do: taskwarrior
- Image viewer: jp2a  
- Video recording/editing: ffmpeg  
- PDF viewer: zathura  
- Shell: ofc vi mode
- Weather: `curl wttr.in/`

# Addons
- Firefox: [Tridactyl](https://github.com/tridactyl/tridactyl)  
- Code: [Vim](https://github.com/VSCodeVim/Vim)  

# TTY
- WM: byobu  
- Video player: mpv
